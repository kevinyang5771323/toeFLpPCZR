# 项目简介

## 核心功能
- 系统管理员管理：用户数据录入、修改、删除
- 药品类型管理：药品类型数据输入、删除、查询、详情、修改
- 药品信息管理：查询、修改、添加药品信息（编号、名称、类型、价格、包装、库存、作用介绍）
- 订单管理：增加、修改、删除订单数据（编号、名称、类型、价格、数量、总价、时间、备注、用户名）
- 采购计划管理：查询、添加、修改、删除采购计划（编号、名称、数量、预算价格、申请时间、备注、工号）
- 供应商管理：录入、查询、编辑、删除供应商信息（编号、名称、地址、货物、联系方式、备注）
- 在线留言管理：数据输入、删除、查询、详情、修改（昵称、头像、联系方式、留言内容、回复内容）

## 适用场景
- 医疗机构
- 药店
- 药品销售公司
- 药品研发企业

## 技术特点
- 使用Python编程语言开发
- 支持前后端分离，易于维护和扩展
- 集成数据库文件，数据管理便捷
- 提供详细文档和部署说明，方便使用和部署
- 包含功能演示视频，便于用户快速上手
